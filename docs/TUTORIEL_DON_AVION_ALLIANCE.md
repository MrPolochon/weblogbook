# Tutoriel : Faire un don d'avion via le système d'alliance

Ce guide explique comment donner un avion à une autre compagnie membre de votre alliance sur le site.

---

## Prérequis

1. **Votre compagnie doit être membre d'une alliance**
   - Si ce n'est pas le cas, un président d'alliance peut vous inviter (section **Alliance** > invitations en attente).

2. **L'option « Don d'avions entre membres » doit être activée**
   - Seul le **Président** ou un **Vice-Président** peut activer cette option.
   - Aller dans **Alliance** > onglet **Paramètres** > cocher **« Don d'avions entre membres »**.

3. **Vous devez être PDG** de la compagnie qui possède l'avion.

4. **L'avion doit être disponible**
   - Au sol, pas en vol
   - Pas en location chez une autre compagnie
   - Pas en maintenance

---

## Étape 1 : Proposer un don

### Depuis Ma Compagnie > Flotte

1. Allez dans **Ma Compagnie** (menu principal).
2. Ouvrez l’onglet **Flotte**.
3. Repérez l’avion que vous souhaitez donner.
4. Utilisez le menu d’actions de l’avion :
   - Si un bouton ou une option **« Donner à un membre »** ou **« Transfert alliance »** est disponible, cliquez dessus.
   - Sinon, utilisez l’option **« Mettre en location »** (icône poignée de main) : dans le menu, vérifiez s’il existe un mode **Don** ou **Transfert alliance** vers les membres de l’alliance.

5. Dans la fenêtre qui s’ouvre :
   - Choisissez le **type de transfert** : **Don** (gratuit).
   - Sélectionnez la **compagnie destinataire** (un membre de votre alliance, autre que la vôtre).
   - Validez la proposition.

> **Note** : Si vous ne voyez pas d’option pour proposer un don, vérifiez que l’option est bien activée dans **Alliance** > **Paramètres** par un dirigeant.

---

## Étape 2 : Côté destinataire — Accepter ou refuser

Le **PDG de la compagnie destinataire** reçoit une notification et doit accepter ou refuser le don.

1. Allez dans **Alliance** (lien dans Ma Compagnie ou menu).
2. Ouvrez l’onglet **Flotte**.
3. Dans **« Transferts en attente »**, vous verrez la proposition de don :
   - Type : **DON**
   - De : compagnie donatrice
   - Vers : votre compagnie

4. Choisissez :
   - **Accepter** : l’avion est transféré définitivement à votre compagnie.
   - **Refuser** : la proposition est annulée.

---

## Types de transferts possibles

Le système d’alliance permet trois types de transferts :

| Type   | Description                          | Prix      |
|--------|--------------------------------------|-----------|
| **Don** | Transfert gratuit                    | 0 F$      |
| **Vente** | Vente entre membres                  | Prix défini par le vendeur |
| **Prêt** | Prêt temporaire (ex. 7 jours)        | Gratuit   |

Chaque type doit être activé séparément dans **Alliance** > **Paramètres** :
- **Don d'avions entre membres**
- **Vente d'avions entre membres**
- **Prêt d'avions entre membres**

---

## Don sans destinataire

Pour un **don**, vous pouvez cocher **« Sans destinataire »** : tout membre de l'alliance peut alors récupérer l'avion (premier arrivé, premier servi). Allez dans **Alliance** > **Flotte** et cliquez sur **Récupérer**.

---

## Vente : marketplace alliance

La **vente** fonctionne comme le Hangar Market, mais réservée aux membres de l'alliance. Vous mettez un prix, tout membre peut acheter. **Un même avion peut être en vente sur le Hangar Market ET dans l'alliance** à des prix différents. Dès qu'il est vendu à un endroit, l'autre annonce est automatiquement annulée.

---

## Résumé du flux

```
[Compagnie A - Donateur]                    [Compagnie B - Destinataire]
         |                                            |
         | 1. Propose un don (Ma Compagnie > Flotte)   |
         |    → Avion X vers Compagnie B               |
         |                                            |
         | 2. Notification reçue                       |
         |                                            |
         |                    3. Alliance > Flotte     |
         |                       → Accepter / Refuser |
         |                                            |
         | 4. Si accepté : avion transféré à B         |
         |    Si refusé : proposition annulée         |
```

---

## Dépannage

- **« Le don d'avions entre membres n'est pas activé »**  
  → Un Président ou Vice-Président doit activer l’option dans Alliance > Paramètres.

- **« Destination pas dans l'alliance »**  
  → La compagnie choisie n’est pas membre de votre alliance.

- **« Avion non trouvé ou pas à vous »**  
  → Vous n’êtes pas PDG de la compagnie propriétaire, ou l’avion n’existe pas.

- **Pas d’option pour proposer un don**  
  → Vérifiez que vous êtes bien PDG et que l’option est activée. Si l’interface ne propose pas encore cette action, contactez un administrateur.
